#### 安装教程

1. 下载HbuilderX
2. 手机USB连接电脑
3. 打开开发者模式, USB调试
4. 打开HbuilderX, 选择 运行 -> 运行到手机或模拟器 -> 选择连接的手机  等待安装

#### 使用说明
将项目拖入[HbuilderX](http://www.dcloud.io/hbuilderx.html),直接运行即可
[uni-app教程](https://uniapp.dcloud.io)
